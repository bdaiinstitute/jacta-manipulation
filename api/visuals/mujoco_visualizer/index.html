class visuals.mujoco_visualizer.MujocoRenderer(*, plant: dexterity.jacta_planner.dynamics.mujoco_dynamics.MujocoPlant | None = None, model_filepath: str | None = None, time_step: float | None = None, collision_geometry_opacity: float | None = None)#

MujocoMeshcatRenderer is a class for rendering the Mujoco trajectories in meshcat.

It parses bodies from Mujoco.MjModel and animates them using minimal coordinates.

Example: ```

renderer = MujocoRenderer(plant = plant, time_step=0.01) for state in state_trajectory:

renderer.render(state)

renderer.show_goal(goal_state) renderer.play() OR renderer.save(“recording.html”)
