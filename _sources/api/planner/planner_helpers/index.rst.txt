.. py:function:: is_object_tilted(sensordata: torch.FloatTensor, tilt_tolerance: float = np.pi / 9.0, y_axis_indexes: slice = slice(24, 27)) -> torch.BoolTensor

   
   Determine if an object is tilted beyond a specified tolerance.

   This function calculates the tilt angle of an object based on its y-axis orientation
   and compares it to a given tolerance to determine if the object is considered tilted.

   :param sensordata: A tensor containing sensor data with ``<frameyaxis>`` sensors enabled.
   :param tilt_tolerance: The maximum allowable tilt angle in radians. Defaults to Ï€/9 (20 degrees).
   :param y_axis_indexes: A slice object representing the indexes of the object's y-axis in the sensor data.

   :returns: *bool* -- True if the object is tilted beyond the tolerance, False otherwise.

.. py:function:: is_object_out_of_reach(sensordata: torch.FloatTensor, reach_tolerance: float = 0.5, dist_indexes: slice = slice(21, 24)) -> torch.BoolTensor

   
   Determine if an object is out of reach based on its distance from the indexed distance sensor.

   :param sensordata: A tensor containing sensor data with ``<framepos>`` sensor enabled.
   :param reach_tolerance: The maximum allowable distance in meters. Defaults to 1.5 meters.
   :param dist_indexes: A slice object representing the indexes of the object's distance in the sensor data.

   :returns: *bool* -- True if the object is out of reach, False otherwise.
