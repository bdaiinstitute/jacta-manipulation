Jacta: A Versatile Planner for Learning Dexterous and Whole-body Manipulation#

Robotic manipulation is challenging and data-driven approaches typically require large amounts of data or expert demonstrations. Therefore, we introduce a motion planner for dexterous and whole-body manipulation tasks. The plannerâ€™s demonstrations can directly be used by reinforcement learning. With this approach, we can efficiently learn policies for complex manipulation tasks, where traditional reinforcement learning alone only makes little progress.

Installation#

Install cmake

sudo apt install cmake

Install mujoco_extensions

git clone https://github.com/bdaiinstitute/jacta-manipulation.git
cd jacta-manipulation
pip install src/mujoco_extensions -vv

Install jacta-manipulation

pip install -e .

Getting started#

python examples/planner/example_notebook.py

Open the visualizer in your browser by clicking on the link in the terminal.

http://localhost:8080/
